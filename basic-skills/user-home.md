# 用户主目录服务设置指南

Synology NAS 的“用户主目录服务（User Home Service）”可以为每个用户自动分配一个独立的“私人储物柜”，只有用户本人和管理员可以访问。

## 什么是用户主目录？

启用后，系统会自动在 `homes` 共享文件夹下为每个用户创建一个以用户名命名的子文件夹（例如 `homes/user1`）。用户登录后，会在文件管理器中看到一个名为 `home` 的虚拟文件夹，实际上指向的就是 `homes/你的用户名`。

## 启用步骤 (DSM 7.x)

1.  **进入控制面板**：使用管理员账号登录 DSM，打开“控制面板”。
2.  **用户与群组**：点击“用户与群组”。
3.  **高级设置**：在页面顶部，点击“高级设置”按钮。
4.  **启用服务**：
    - 向下滚动找到“用户主目录”部分。
    - 勾选“启用用户主目录服务”。
    - 选择一个存储空间（例如“存储空间 1”）作为主目录的存储位置。
5.  **应用设置**：点击“应用”保存。

## 注意事项

- **homes 与 home 的区别**：
    - `homes`：这是物理存在的共享文件夹，包含所有用户的主目录。通常只有管理员可见。**请勿修改此文件夹的权限，否则可能导致用户无法访问自己的主目录。**
    - `home`：这是每个用户登录后看到的虚拟入口，指向自己的私人目录。
- **回收站**：建议为 `homes` 文件夹启用回收站功能，防止误删文件。

## 常见用途

- **Synology Drive**：启用主目录后，每个用户都会有自己的“我的文件”区域。
- **Synology Photos**：用户可以拥有自己的“个人空间”来存储照片。
