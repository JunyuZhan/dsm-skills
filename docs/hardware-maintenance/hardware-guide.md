# 硬件选购与维护指南

NAS 不仅仅是软件，硬件的稳定性和维护同样重要。

## 1. 硬盘选购：避坑指南
- **CMR vs SMR**：**绝对不要买 SMR (叠瓦式) 硬盘**做 RAID。SMR 硬盘在 RAID 重建时极易掉盘，导致数据全丢。
    - **推荐**：希捷酷狼 (IronWolf)、西数红盘 Plus (Red Plus)、企业盘 (Exos/Ultrastar)。
    - **避坑**：西数红盘 (不带 Plus 的型号)、普通台式机蓝盘/绿盘。
- **噪音**：企业盘 (16TB+) 炒豆子声音很大。如果放在卧室，建议选择 5400 转的 NAS 专用盘或 4TB 以下的静音盘。

## 2. RAID 模式选择：SHR 的优势
- **SHR (Synology Hybrid RAID)**：
    - **优势**：允许不同容量的硬盘混用，空间利用率最大化。未来升级时，只需换掉两块硬盘即可扩容。
    - **原理**：底层是 LVM + mdadm，自动划分 RAID 1/5 区域。
- **RAID 5**：
    - **限制**：所有硬盘必须同容量，否则按最小盘计算。
    - **建议**：除非你有 4 块完全一样的硬盘且以后不打算一块块升级，否则**无脑选 SHR**。

## 3. NVMe SSD 存储池 (黑科技)
- **官方限制**：Synology 大部分机型只允许 NVMe 做缓存，不能做存储池 (Volume)。
- **破解**：使用 GitHub 上的 `Synology_enable_M2_volume` 脚本。
- **优势**：
    - 读写速度突破 SATA 限制 (可达 1000MB/s+)。
    - 运行 Docker 和虚拟机飞快。
    - **注意**：建议使用企业级 SSD 或高 TBW 的型号，并做好散热。

## 4. UPS 不间断电源：NAS 的保命符
- **必要性**：RAID 写入时断电会导致文件系统损坏。
- **配置**：
    - 购买支持 USB 通讯的 UPS (APC/CyberPower/Eaton)。
    - 连接 NAS 后，在“硬件和电源”中开启 UPS 支持。
    - **NUT 服务器**：如果你有多台设备 (NAS + PC + 路由器)，可以将 Synology 设为 NUT Server，其他设备作为 Client 同步关机。

## 5. 内存升级：ECC vs Non-ECC
- **官方建议**：使用官方内存。但太贵。
- **第三方内存**：三星、以及国产大厂的笔记本内存通常兼容。
- **ECC**：如果你是 Plus 或 XS 系列 (如 DS1621+, DS923+)，支持 ECC 内存。ECC 能纠正位翻转错误，对 ZFS/Btrfs 数据完整性有好处，强烈建议使用 ECC 内存 (虽然贵一点)。

## 6. 散热与静音改造
- **风扇**：原厂风扇通常噪音大。可以更换 **Noctua (猫头鹰)** 风扇。
    - **注意**：部分机型检测不到猫扇转速会报警，需要修改 `/usr/syno/etc.defaults/scemd.xml` 配置文件。
- **防尘**：Synology 侧面进风口通常没有防尘网。可以购买磁吸防尘网贴在侧面 LOGO 处，减少内部积灰。

## 7. 硬盘坏道检测与处理
- **SMART 检测**：每半年做一次“完整检测” (Extended Test)。
- **坏道预警**：
    - 如果 `Reallocated Sector Count` (05) 或 `Current Pending Sector` (C5) 不为 0，立刻更换硬盘！不要抱有侥幸心理。
    - 在保修期内直接申请 RMA。

## 8. 万兆网络 (10GbE) 升级
- **原生支持**：DS1621+ 等机型有 PCIe 插槽。
- **USB 转换器**：对于无 PCIe 插槽的机型 (如 DS920+)，可以使用基于 Aquantia AQC111U 芯片的 5G/2.5G USB 网卡，配合 GitHub 上的驱动包实现 2.5G/5G 网络。

## 9. 硬盘休眠问题排查
- **现象**：硬盘频繁唤醒，无法休眠。
- **排查**：
    - 拔掉网线，如果能休眠，说明是网络唤醒 (QuickConnect/DDNS)。
    - 停用 Docker，容器日志是休眠杀手。
    - 使用 `cat /var/log/messages | grep wake` 查看唤醒原因。

## 10. 清灰维护
- **频率**：每年一次。
- **步骤**：
    1.  关机，拔电源。
    2.  抽出硬盘架 (标记好顺序，千万别插乱)。
    3.  用压缩空气罐吹主板和风扇。
    4.  **切记**：不要用吸尘器对着风扇猛吸，可能会产生静电击穿主板。
