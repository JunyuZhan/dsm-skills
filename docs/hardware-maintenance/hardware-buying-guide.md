# 硬件选购与避坑指南 (2025 版)

NAS 只有两个核心：**数据安全**和**服务稳定**。选购硬件时，不要只看参数，要看“稳不稳”。

## 1. 品牌 NAS 选购策略

### 群晖 (Synology)
*   **Plus 系列 (+) **：**闭眼买**。支持 Docker、虚拟机、扩展柜。
    *   **DS224+**：双盘位性价比之王。J4125 处理器，支持硬解。适合家用入门。
    *   **DS923+ / DS723+**：AMD R1600 处理器。**优点**：支持 ECC 内存，支持万兆扩展。**缺点**：无核显（无法硬件转码）。适合纯存储、Docker、虚拟机玩家。
    *   **DS423+**：J4125 处理器。四盘位硬解神器。适合重度影音玩家 (Plex/Emby)。
*   **Play / J 系列**：**慎买**。ARM 架构，不支持 Docker，不支持 Btrfs (部分支持但性能差)。只适合纯存文件。

### 威联通 (QXZ) / 极空间 / 绿联
*   **优势**：硬件配置极高（N100/N305 满血版），价格便宜。
*   **劣势**：系统稳定性不如 DSM。
*   **玩法**：买来刷黑群晖（需自行解决引导问题）。

## 2. DIY NAS (黑群晖) 硬件推荐

如果你追求极致性价比，DIY 是不二之选。

### 主板/CPU 平台
*   **J4125 (经典)**：兼容性最好（DS920+ 原型），核显驱动完美。4K 转码够用。
*   **N100 (新贵)**：性能比 J4125 强 40%，但核显驱动在旧版引导下可能有问题（推荐 RR/ARC 引导）。
*   **N305 / 8505**：8 核高性能，适合跑大量 Docker 和虚拟机。
*   **i3-8100 / i5-8500 (旧平台)**：桌面级 CPU，性能吊打低功耗 U，但功耗较高。

### 网卡选择 (重中之重)
黑群晖最怕网卡不识别。
*   **Intel**：I210, I211, I219, I225-V (B3步进), I226-V。**推荐 I226-V**，原生支持 2.5G。
*   **Realtek**：RTL8125B (2.5G)。兼容性不错。
*   **Mellanox**：ConnectX-3 (万兆光口)。免驱神卡。

## 3. 硬盘选购：拒绝 SMR

**铁律：严禁使用 SMR (叠瓦式) 硬盘组 RAID。** SMR 硬盘在 RAID 重建（大量随机写入）时，速度会掉到几 KB/s，导致掉盘、RAID 损毁。

### 推荐红榜 (CMR)
*   **4TB**: 希捷酷狼 (IronWolf), 西数红盘 Plus (Red Plus, WD40EFPX)。
*   **8TB**: 希捷银河 (Exos 7E8), 西数 HC320 (空气盘，噪音大但稳)。
*   **16TB+**: 希捷银河 X16/X18, 西数 HC550。企业盘虽然炒豆子声音大，但故障率极低。

### 避坑黑榜 (SMR)
*   **西数红盘 (WDx0EFAX)**：不带 Plus 的老款红盘。
*   **希捷酷鱼 (BarraCuda)**：大部分是 SMR。
*   **2.5寸笔记本盘**：99% 是 SMR，绝对不要用。

## 4. 固态硬盘 (SSD)

*   **SATA SSD**：用于存储池。推荐三星 870 EVO (稳) 或企业级 SATA (Intel S4510)。
*   **NVMe SSD**：用于缓存或存储池。
    *   **缓存**：读写极其频繁，必须买高 TBW (写入寿命) 的盘。推荐致态 TiPlus7100 (无缓低温), 凯侠 RC20。
    *   **避坑**：不要买 QLC 颗粒的 SSD 做缓存，寿命很快耗尽。

## 5. 网络设备

NAS 再快，网线不行也白搭。

*   **交换机**：推荐 2.5G 傻瓜交换机 (兮克/TP-LINK)，几十块钱就能跑满机械硬盘速度。
*   **网线**：**超五类 (Cat5e)** 短距离内完全能跑万兆。不需要非得买七类/八类线（又硬又难以此）。**六类 (Cat6)** 是性价比首选。

## 6. 必买配件

*   **UPS (不间断电源)**：几百块买个安心。推荐 APC BK650 或雷神。
*   **防尘网**：NAS 放在地上就是吸尘器。买个磁吸防尘网贴在进风口。
*   **减震垫**：机械硬盘共振会有低频噪音。买点 3M 硅胶垫贴在 NAS 脚垫下。
