# 硬件周边选购指南

NAS 只是一个核心，要发挥它的最大威力，还需要周边硬件的配合。以下指南基于群晖 DSM 兼容性及社区长期验证的经验。

## 1. 不间断电源 (UPS)

**这是 NAS 最重要的“保险”！** 突然断电不仅可能导致数据丢失，更可能物理损坏硬盘磁头。

*   **工作原理**：市电断开 -> UPS 供电 -> USB 通知 NAS -> NAS 进入安全模式（停止读写，卸载存储池） -> UPS 耗尽关机。
*   **选购要点**：
    *   **必须带有 USB 通讯接口**。没有 USB 的 UPS 对 NAS 毫无意义。
    *   **品牌推荐**：
        *   **APC (施耐德)**：BK650M2-CH (一代神机，免驱完美兼容)。
        *   **CyberPower (雷神)**：性价比较高。
        *   **山特 (SANTAK)**：注意要买带 USB 的型号（如 TG-BOX 850）。
*   **配置**：
    *   连接 USB 线到 NAS。
    *   控制面板 > 硬件和电源 > 不间断电源 > 勾选“启用 UPS 支持”。
    *   **断电设定**：建议设置为“进入安全模式之前的等待时间：5 分钟”。不要勾选“不断电系统关机”，因为如果市电恢复了，UPS 可能不会自动唤醒 NAS，而“安全模式”可以确保市电恢复后 NAS 自动恢复。

## 2. 万兆网卡 (10GbE) 与 2.5G

机械硬盘阵列的读写速度轻松突破 300MB/s，千兆网络 (100MB/s) 成了最大瓶颈。

*   **官方卡**：
    *   **E10G18-T1 / E10G21-F2**：贵，但兼容性 100%，支持 SR-IOV（虚拟机性能更好）。
*   **第三方卡 (PCIe 插槽机型)**：
    *   **Mellanox ConnectX-3 (CX341A)**：洋垃圾神卡，光口，几十块钱，群晖免驱直接认。
    *   **Aquantia AQC107**：电口万兆的主流方案，部分型号免驱。
*   **USB 转 2.5G/5G (无 PCIe 插槽机型)**：
    *   对于 DS920+, DS224+ 这种没有扩展槽的机器，可以使用 USB 网卡。
    *   **芯片**：推荐 RTL8156B (2.5G)。
    *   **驱动**：DSM 7 不自带驱动，需要由 [bb-qq](https://github.com/bb-qq/r8152) 提供的驱动包。

## 3. 内存 (RAM) 升级

Docker 和虚拟机是内存大户。

*   **官方内存**：极贵，但保修无忧。
*   **第三方内存**：
    *   **三星 / 镁光 / 海力士**：兼容性最好。
    *   **规格**：
        *   DDR4 2666 ECC / Non-ECC SO-DIMM (笔记本内存条)。
        *   **注意 ECC**：如果你的 CPU 支持 ECC（如 Ryzen 系列或 Xeon），强烈建议买纯 ECC 内存（不是 REG ECC），能极大提高数据稳定性。
    *   **容量**：虽然官网写最大支持 6GB (2+4)，但实测 DS920+/DS923+ 等机型都可以完美支持 16GB 甚至 32GB 内存。

## 4. NVMe SSD 缓存与存储池

*   **作缓存**：
    *   加速小文件读写（如 Photos 缩略图、Web 服务器）。
    *   **寿命焦虑**：缓存对 SSD 写入量极大，建议购买高 TBW 的企业级 SSD，或者在 DSM 中只开“只读缓存”（RAID 0 安全，坏了不丢数据）。
*   **作存储池**：
    *   DSM 7.2+ 支持将 NVMe 建立为存储池（Volume），适合跑 Docker 和虚拟机。
    *   **限制**：官方只允许群晖自有品牌 SSD 创建存储池。
    *   **破解**：社区有脚本 (`007revad/Synology_HDD_db`) 可以解锁第三方 SSD 创建存储池的限制。

## 5. 硬盘扩展柜 vs 外接硬盘盒

当盘位满了怎么办？

*   **官方扩展柜 (DX517)**：
    *   **优点**：通过 eSATA 连接，系统视为内部存储池，可以跨柜组 RAID（虽然不推荐）。
    *   **缺点**：价格几乎能买个新 NAS 了。
*   **USB 硬盘盒**：
    *   **用途**：只能作为“外接存储”挂载，**不能**加入 RAID，**不能**安装套件（除非魔改）。
    *   **适合**：冷备份、临时数据交换。
    *   **推荐**：世特力、优越者等多盘位柜，支持 USB 3.0/3.1。

## 6. 智能家居网关 (Zigbee/Z-Wave Dongle)

如果你在 NAS 上跑 Home Assistant。

*   **推荐**：
    *   **Sonoff Zigbee 3.0 USB Dongle Plus (P版)**：基于 CC2652P 芯片，信号强，兼容性好。
    *   **ConBee II**：老牌经典，德系品质。
*   **坑点**：DSM 7 默认隐藏了 USB 串口驱动。插入后可能不识别。
    *   **解决**：需要手动加载 `cp210x.ko` 或 `ch341.ko` 驱动（可通过搜索相关脚本解决）。
    *   **最佳实践**：不要直插 NAS USB 口，**用一根 USB 2.0 延长线**，避免 USB 3.0 硬盘的信号干扰 Zigbee 信号。
