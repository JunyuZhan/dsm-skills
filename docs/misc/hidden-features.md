# 冷知识与隐藏功能 (Misc & Hidden Features)

DSM 里还有很多鲜为人知的小秘密。

## 1. 网页版 DSM 快捷键
- **Ctrl + F**：全局搜索。
- **Ctrl + Alt + Space**：切换输入法（如果和本地冲突，可在个人设置中修改）。
- **Shift + 拖动**：多选文件。
- **Alt + 双击**：查看文件属性。

## 2. 桌面模式切换
- **技巧**：默认是窗口模式。你可以把 DSM 设置为“平板模式”（全屏应用列表），更适合触屏设备操作。

## 3. 自定义任务栏
- **技巧**：你可以把常用的套件拖动到顶部任务栏，固定在那里，方便随时打开。

## 4. 音频输出到 USB 音箱
- **技巧**：虽然 DSM 7.0 移除了对 USB 声卡的部分支持，但 Audio Station 仍然可以通过 USB 扬声器播放音乐（需兼容的硬件）。

## 5. 生成共享链接的有效期
- **技巧**：在 File Station 分享文件时，务必设置“有效期”和“访问次数”，防止链接泄露后永久有效。

## 6. 文件请求 (File Request)
- **场景**：你要收集全班同学的作业，或者婚礼上收集大家的照片。
- **技巧**：在 File Station 中创建一个文件夹 > 右键 > 创建文件请求。发送链接给他人，他们就能直接上传文件到这个目录，但看不到目录里其他人的文件。

## 7. 文本编辑器 (Text Editor)
- **技巧**：DSM 自带一个简易的代码编辑器。支持语法高亮（Python, PHP, HTML 等）。适合临时修改配置文件。

## 8. 日志中心 (Log Center)
- **技巧**：不仅仅看日志。你可以设置“关键字通知”。例如，当日志中出现“I/O error”或“System fan failed”时，立即发邮件报警。

## 9. 安全顾问的自定义检查
- **技巧**：你可以配置安全顾问的扫描计划和规则。比如你确实需要开启 SSH，可以把 SSH 相关的警告忽略掉，以免每次都报红。

## 10. 释放内存的“野路子”
- **技巧**：如果系统缓存（Cached）占用太高导致某些应用起不来（虽然理论上会自动释放），可以 SSH 登录执行 `sync; echo 3 > /proc/sys/vm/drop_caches` 手动清理缓存（慎用，会暂时降低系统性能）。
