# Note Station：被低估的生产力工具

在 Notion、Obsidian 满天飞的时代，群晖原生的 **Note Station** 依然有一席之地。它完全私有、免费、且与 DSM 系统深度集成。

## 1. Web Clipper：网页剪藏神器

这是 Note Station 最核心的杀手锏，体验堪比 Evernote（印象笔记）巅峰时期。

1.  **安装**：在 Chrome/Edge 商店搜索并安装 **Synology Web Clipper** 扩展。
2.  **使用**：看到好的技术文章、食谱、教程，点击扩展图标。
3.  **模式**：
    *   **简化内容**：自动去除广告、侧边栏，只保留正文和图片（阅读体验极佳）。
    *   **完整页面**：所见即所得。
    *   **截图**：截取部分区域。
4.  **保存**：直接保存到 Note Station 指定笔记本。即使原网站倒闭了，你的资料永远都在 NAS 里。

## 2. 待办事项 (To-do List)

Note Station 不仅仅是笔记，还是一个轻量级的 GTD 工具。

*   **创建任务**：在笔记中可以直接插入待办事项。
*   **独立视图**：左侧菜单有 **待办事项** 视图，汇总所有笔记里的任务。
*   **提醒**：设置截止日期和提醒时间，DSM 桌面会弹窗提醒，手机 DS note 也会推送。
*   **星标**：将重要任务标记为星标，优先处理。

## 3. 演示模式 (Presentation)

临时要开会，来不及做 PPT？
*   打开一篇笔记。
*   点击右上角的 **演示** 按钮。
*   Note Station 会自动把笔记标题转为 PPT 标题，正文转为内容，支持图片轮播。
*   虽然没有 Keynote 华丽，但用于内部分享、代码展示绰绰有余。

## 4. 笔记加密与版本控制

*   **加密**：
    *   **单篇加密**：右键笔记 > 加密。设置密码。
    *   **查看**：每次查看需要输入密码。
    *   **场景**：保存银行卡号、密码本、日记。
*   **版本历史**：
    *   误删了一段话？点击工具栏的 **历史记录**。
    *   可以查看并还原到任意一个历史时间点。

## 5. 导入与导出

*   **导入**：支持从 **Evernote (.enex)** 导入数据。如果你想逃离 Evernote 的付费限制，Note Station 是最佳落脚点。
*   **导出**：支持导出为 `.nsx` (群晖专用格式) 备份，或者导出为 PDF 打印。

## 6. 手机端 DS note

*   **离线访问**：可以将常用笔记本下载到手机，没网也能看。
*   **快速记录**：支持语音速记、拍照笔记。
*   **快捷方式**：Android 支持添加桌面小组件。
