# 基础入门

欢迎开始你的 Synology NAS 之旅。本章节涵盖了 DSM 系统最核心的基础设置，打好基础才能让 NAS 稳定运行。

## 目录

*   [**系统设置遗珠**](system-gems.md): 那些鲜为人知但极其好用的开关（LED 亮度、登录界面美化、UPS 联动）。
*   [**用户主目录服务**](user-home.md): 理解 `/home` 与 `/homes` 的区别，为每个用户分配私有空间。
*   [**存储管理器深度解析**](storage-manager.md): **(必读)** RAID 选什么？Btrfs 好在哪？如何进行数据清洗？
*   [**系统安全性提升**](security.md): 2FA、防火墙、自动封锁，保卫你的数据。
*   [**通知中心与日志审计**](notifications-logs.md): 让 NAS 学会“说话”，第一时间通过微信/邮件告诉你发生了什么。
*   [**内存检查与优化**](memory-check.md): 资源监控器的进阶用法，排查卡顿原因。
