# 基础入门
2
3欢迎开始你的 Synology NAS 之旅。本章节涵盖了 DSM 系统最核心的基础设置，打好基础才能让 NAS 稳定运行。
4
5## 目录
6
7*   [**文件 Station 深度指南**](file-station.md): 掌握快捷键、后台任务和高级搜索，像操作本地电脑一样管理文件。
8*   [**文件服务深度调优**](file-services.md): SMB 多通道、NFS 权限陷阱，让传输速度翻倍。
9*   [**存储管理器深度解析**](storage-manager.md): **(必读)** 详解 SHR 优势、硬盘更换策略 (Replace Drive) 和 NVMe 存储卷。
*   [**系统安全性提升**](../network/security-hardening.md): 防火墙黄金法则、免密登录与账户保护，构建纵深防御体系。
*   [**权限体系底层逻辑**](permission-logic.md): 彻底搞懂 Linux 权限与 ACL 的关系，解决“禁止访问”的千古之谜。
12*   [**用户与群组管理**](identity-management.md): RBAC 最佳实践，为什么永远只给群组赋权？
13*   [**通知中心深度指南**](notification-center.md): 配置 Webhook (钉钉/微信) 和短信报警，告别邮件延迟。
14*   [**日志中心实战**](log-center.md): 搭建 Syslog Server 收集路由器日志，使用 SSH 查阅内核报错。
15*   [**电源管理 (UPS & WOL)**](power-management.md): UPS 选购与配置、网络 UPS 服务器、远程唤醒全攻略。
16*   [**内存诊断与 Swap 调优**](memory-check.md): 开启内存自检模式，调整 Swappiness 解决硬盘噪音。
17*   [**资源监控深度指南**](resource-monitor.md): 看懂 I/O Wait，排查系统卡顿的真凶。
18*   [**索引服务深度解析**](indexing-service.md): 解决 SSH 导入文件不索引的问题，优化 CPU 占用。
19*   [**WebDAV 远程挂载**](../network/webdav-guide.md): 解决中文乱码，替代 SMB 实现公网流畅播放。
20*   [**Cloud Sync 深度指南**](cloud-sync.md): 客户端加密上传百度网盘，保护隐私。
21*   [**系统设置遗珠**](system-gems.md): 任务栏自定义、SSH 配色、LED 调度等 10 个隐藏小技巧。
