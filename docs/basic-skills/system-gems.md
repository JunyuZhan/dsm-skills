# 系统设置遗珠 (System Gems)：你可能错过的 10 个小功能

除了核心功能，DSM 还有很多隐藏在角落里的小开关，开启它们能大幅提升体验。

## 1. 任务栏自定义 (Taskbar)
不用每次都去“主菜单”里找套件。
*   **操作**：打开任意套件（如 Docker），右键点击任务栏上的图标 > **固定到任务栏**。
*   **排序**：直接拖动任务栏图标调整顺序。

## 2. 桌面小部件 (Widgets)
像 Windows 侧边栏一样监控系统。
*   **操作**：点击桌面右上角的 **小部件** 图标。
*   **推荐**：
    *   **资源监控**：实时看 CPU/内存。
    *   **存储空间**：看硬盘满了没。
    *   **连接用户**：看谁在线。

## 3. SSH 终端配色
默认的 SSH 是黑白的，很难看。
*   **操作**：SSH 登录后，编辑 `~/.bashrc`。
*   **添加**：
    ```bash
    export PS1="\[\033[01;32m\]\u@\h\[\033[00m\]:\[\033[01;34m\]\w\[\033[00m\]\$ "
    alias ll='ls -alF --color=auto'
    ```
*   **生效**：`source ~/.bashrc`。现在你的终端有颜色了，文件夹和文件一目了然。

## 4. 禁用哔声 (Beep Control)
半夜重启 NAS，“滴”的一声能把全家吵醒。
*   **路径**：控制面板 > 硬件和电源 > 常规 > 哔声控制。
*   **建议**：
    *   **取消勾选**：系统就绪时（开机完成）。
    *   **保留勾选**：风扇故障、存储空间降级（这些是救命的报警）。

## 5. LED 灯光调度
NAS 放在卧室，晚上蓝灯太亮？
*   **路径**：控制面板 > 硬件和电源 > 常规 > LED 亮度控制。
*   **操作**：点击 **设置日程表**。
*   **方案**：
    *   08:00 - 22:00：全亮。
    *   22:00 - 08:00：**关闭** (Off)。

## 6. 查找序列号与 MAC 地址
申请保修或配置路由器 DHCP 绑定时需要。
*   **路径**：控制面板 > 信息中心 > 网络。
*   **技巧**：如果你有多个网口，每个网口都有独立的 MAC 地址。WOL 唤醒通常绑定 LAN 1 的 MAC。

## 7. 释放 80/443 端口
想用 Docker 跑 Nginx，却发现 80 端口被 Web Station 占用了？
*   **原理**：DSM 默认保留 80/443 给 Web Station 和反向代理。
*   **解决**：不要试图杀进程。
*   **正确做法**：利用 **Macvlan** 网络给 Docker 容器分配独立的 IP 地址。这样容器就拥有了自己的 80 端口，不和 NAS 抢。

## 8. 个人化桌面壁纸
每个人登录看到不一样的壁纸。
*   **路径**：点击右上角头像 > 个人设置 > 桌面。
*   **区别**：
    *   **控制面板 > 登录门户**：改的是**登录前**的画面（所有人一样）。
    *   **个人设置 > 桌面**：改的是**登录后**的画面（你是风景，他是动漫）。

## 9. 快速释放内存
系统慢了，想清空缓存？
*   **原理**：Linux 会把空闲内存用作缓存 (Cache)，这通常是好事。但如果你想测试真实内存占用：
*   **SSH 命令**：
    ```bash
    sync; echo 3 > /proc/sys/vm/drop_caches
    ```
*   **注意**：这会导致系统瞬间变卡（因为要重新从硬盘读取数据），仅用于测试。

## 10. 重新索引 (Re-index)
相册里有张图删了缩略图还在？
*   **操作**：控制面板 > 索引服务 > 媒体索引 > **重建索引**。
*   **范围**：这会重建所有（视频/图片/音乐），耗时极长。
*   **针对性**：如果是 Synology Photos，去 Photos 设置里点“重建索引”更精准。
