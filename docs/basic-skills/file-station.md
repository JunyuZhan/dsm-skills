# File Station 深度指南：不仅是文件管理

File Station 是群晖 DSM 最基础的套件，但 90% 的用户只把它当作“网页版资源管理器”。实际上，它集成了**云盘挂载**、**文件收集**、**全文搜索**、**ACL 权限管理**等高级功能。

## 1. 远程连接：挂载云盘与异地 NAS

你可以直接在 File Station 中挂载 Google Drive、OneDrive、Dropbox，或者另一台 NAS (SMB/WebDAV/FTP)，实现数据“云搬运”。

### 挂载云盘
1.  打开 **File Station**。
2.  点击 **工具** > **远程连接** > **连接设置**。
3.  选择 **Google Drive** 或 **OneDrive** 等。
4.  按照弹窗进行 OAuth 授权。
5.  **效果**：你的云盘会作为一个文件夹出现在左侧列表中。你可以直接把 NAS 里的电影复制到云盘，或者把云盘里的照片备份回 NAS。

### 挂载异地 NAS (CIFS/SMB)
1.  **工具** > **装载远程文件夹** > **CIFS 共享文件夹**。
2.  **文件夹**: `\\192.168.2.100\video` (异地 NAS 地址)。
3.  **账号密码**: 输入异地 NAS 的账号。
4.  **装载到**: 选择本地的一个空文件夹。
5.  **应用场景**：两台 NAS 之间传输几十 TB 数据，直接挂载后复制粘贴，利用 NAS 之间的带宽，不需要经过电脑中转。

## 2. 文件请求 (File Request)：优雅地收文件

你是老师要收作业？还是剪辑师要收素材？不要再让别人加你微信传文件了。

1.  在 File Station 中选择一个文件夹（用于存放收到的文件）。
2.  **右键** > **创建文件请求**。
3.  **设置**：
    *   **密码保护**：防止陌生人上传。
    *   **有效期**：设置截止日期。
    *   **你的姓名**：让上传者知道是发给谁的。
4.  **生成链接**：把链接发给对方。
5.  **对方视角**：打开链接 -> 输入名字 -> 拖入文件 -> 上传。**对方只能上传，看不到文件夹里已有的其他文件**。保护隐私。

## 3. 全文搜索与 OCR

找不到半年前存的 PDF 合同了？只记得内容里有“甲方”两个字？

1.  **安装 Universal Search**：确保已安装并运行。
2.  **建立索引**：
    *   打开 Universal Search > **偏好设置** > **文件索引**。
    *   点击 **索引文件夹列表** > **新增**。
    *   选择你的文档文件夹，勾选 **文档** 类型。
    *   **关键**：如果你的 NAS 性能足够（X86 架构），可以勾选“索引文件内容”。
3.  **搜索**：
    *   在 File Station 右上角搜索框输入关键词。
    *   它不仅搜文件名，还能搜 **Word、Excel、PDF** 里面的文字内容。

## 4. ACL 权限深度管理

Linux 的 `rwx` 权限不够用？DSM 使用的是 Windows 风格的 ACL 权限。

### 场景：允许查看但禁止下载
你想分享一个文件夹给员工看，但怕他拷走？

1.  **右键文件夹** > **属性** > **权限**。
2.  **新增** > 选择用户（如 `staff`）。
3.  **权限编辑器**：
    *   **类型**：允许。
    *   **读取**：勾选“遍历文件夹/执行文件”、“列出文件夹/读取数据”、“读取属性”。
    *   **写入**：全部不勾选。
    *   **删除**：全部不勾选。
    *   **高级**：**不勾选**“下载文件”（这通常需要配合 WebDAV/File Station 设置，SMB 协议很难完全禁止读取但不下载，但在 File Station 网页版有效）。

### 权限继承与显式拒绝
*   **继承**：子文件夹默认继承父文件夹的权限。如果不想继承，可以勾选“将权限应用到子文件夹”来强制覆盖。
*   **拒绝优先**：如果 User A 属于 Group A（允许读）和 Group B（拒绝读），那么**拒绝生效**。

## 5. 虚拟驱动器 (ISO 挂载)

下载了 Windows 安装镜像 ISO？不需要解压。
1.  **右键 ISO 文件** > **装载虚拟驱动器**。
2.  选择一个挂载点。
3.  你可以直接读取 ISO 里的内容，甚至通过 PXE 网络启动来安装这个系统。
