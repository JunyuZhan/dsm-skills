# 其他黑群晖引导方式 (ARC / TCRP)

除了目前最推荐的 RR (Redpill Recovery) 引导，黑群晖领域还有其他几个非常优秀的引导项目。了解它们可以帮助你在遇到特殊硬件兼容性问题时找到替代方案。

## 1. 引导发展简史

*   **Jun's Loader (1.02b/1.03b/1.04b)**: 黑群晖的鼻祖。支持 DSM 6.x，稳定性极高，但已不支持 DSM 7.x，且对新硬件支持较差。目前**不推荐**。
*   **RedPill (红丸)**: 这是一个开源项目，旨在让 DSM 7.x 运行在非官方硬件上。它是所有现代引导（RR, ARC, TCRP）的基石。
*   **TCRP (TinyCore RedPill)**: 基于 TinyCore Linux 的红丸引导制作工具。早期的 DSM 7 引导都需要手动敲命令制作，门槛较高。
*   **ARPL (Automated Redpill Loader)**: 巴西大神制作的自动化引导，彻底降低了门槛。后来作者停止更新，项目分叉为 RR 和 ARC。

## 2. ARC Loader (推荐备选)

**ARC** 是目前与 RR 并驾齐驱的顶级引导，由 `AuxXxilium` 开发。

### 核心特点
*   **硬件兼容性之王**：ARC 对 SAS 阵列卡、SCSI 硬盘、万兆网卡等企业级硬件的支持极其完善。如果你的机器上有 RR 识别不出的硬件，试试 ARC。
*   **Arc Control 套件**：独家功能。安装后会在 DSM 中增加一个“Arc Control”套件，可以在 DSM 系统内直接更新引导、修改参数，无需重启进引导界面。
*   **更新极快**：通常在 Synology 发布新版 DSM 的几天甚至几小时内就会跟进支持。

### 安装简述
1.  **下载**：访问 [ARC GitHub](https://github.com/AuxXxilium/arc/releases) 下载最新镜像（支持 img/vmdk 等格式）。
2.  **写盘**：使用 Rufus 将镜像写入 U 盘。
3.  **启动**：BIOS 设置 U 盘启动。
4.  **配置**：
    *   等待屏幕显示 IP 地址和 Web 端口（通常是 `http://IP:7681`）。
    *   浏览器访问该地址，进入 Arc 配置界面。
    *   **Choose Model**：选择型号（推荐 SA6400 / DS920+）。
    *   **Build**：编译引导。
5.  **安装 DSM**：重启后，通过 `finds.synology.com` 搜索并安装系统。

### 与 RR 的对比

| 特性 | RR (Redpill Recovery) | ARC (Arc Loader) |
| :--- | :--- | :--- |
| **开发者** | 国人 (wjz304 等) | 国际团队 (AuxXxilium) |
| **语言** | 中文支持完美 | 英文为主 (部分有汉化) |
| **易用性** | ⭐⭐⭐⭐⭐ (傻瓜式) | ⭐⭐⭐⭐ (选项极其丰富) |
| **硬件支持** | 优秀 (主流硬件) | **极致** (各种冷门卡、SAS 盘) |
| **特色功能** | 插件系统 (Addons) | Arc Control (DSM 内管理) |
| **推荐人群** | **90% 的用户** | **硬件特殊、喜欢折腾的用户** |

## 3. TCRP (TinyCore RedPill)

**TCRP** 是更底层的引导制作工具，由 `pocopico` 维护。

*   **适用人群**：Linux 极客、开发者、需要深度定制内核参数的用户。
*   **特点**：
    *   没有 Web 界面，全程 SSH 命令行操作。
    *   需要手动下载扩展包 (`./rploader.sh extension add ...`)。
    *   灵活性最高，但也是最容易出错的。
*   **现状**：对于普通用户，TCRP 的功能已经被 RR/ARC 完美封装了，除非你想学习引导原理，否则**不推荐**日常使用。

## 4. 总结与建议

1.  **首选 RR**：如果你是第一次装黑群晖，或者你的硬件是常见的 Intel/AMD 平台，直接用 RR。中文界面友好，插件丰富，足够稳定。
2.  **备选 ARC**：如果你用的是服务器拆机件（如 LSI 阵列卡、SAS 硬盘），或者 RR 无法驱动你的网卡，请尝试 ARC。
3.  **避坑**：不要再去找几年前的“Jun's Loader 1.04b”教程了，那是 DSM 6 时代的产物，无法安装 DSM 7.2。
